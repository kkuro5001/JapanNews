# JapanNewsAi

## 概要
日本の最新ニュースをGNews APIを利用してニュースを収集し、ユーザーの関心に合わせてパーソナライズ表示するニュースフィードアプリで、FlaskをReactで構成されています。

## 主な機能
- 日本の最新ニュースを取得して表示
- React と Flask の連携（CORS対応）
- ログイン時にメールにリンクを送信しクリックするとログイン完了。(できているはず)
- API キーを `.env` で安全に管理
- 今後の拡張予定：ユーザー学習・要約機能 など

## 記録
